有线——树莓派设置静态IP

发表于 2022-08-13 更新于 2023-08-28 分类于 Learn ， Raspberry Pi
本文字数： 946 阅读时长 ≈ 2 分钟

  记录笨蛋笔者设置树莓派静态IP的过程。

起因

  为了防止路由器分配给树莓派的地址总是变来变去，笔者在网上搜索了如何设置树莓派的静态IP地址。

  然而，不论笔者如何设置树莓派的IP静态地址，再次打开时地址还是会变化。

原因

  笔者终于发现了，大多数教程中设置eth0是设置有线网络的静态IP地址，而笔者之前一直用WIFI连接树莓派！

  再强调一遍，此教程是教有线网络的树莓派静态IP地址设置。

  我还真是个笨蛋呐。

设置过程

  1.若要修改静态IP地址，修改 /etc/dhcp.conf 中的配置文件。（DHCP配置文件）

  sudo vim /etc/dhcpcd.conf

  2.在打开的文件中，找到下图所示片段，并更改。

  

  按下i键，进入编辑模式。
  第一行就写 interface eth0即可。
  第二行就在static ip_address后写出来自己想要设置的静态IP地址即可，笔者这里设置了192.168.0.111/24。基本就改111那个部分，前三个部分树莓派连接的路由器怎么写的这里就怎么写。
  第三行写树莓派连接上的路由器地址，可以通过ifconfig指令查看。路由器地址可以通过看路由器上贴的纸来实现。
  第四行写的是域名服务器地址，一般写114.114.114.114 8.8.8.8即可。

  按下esc键后，按下:wq这三个键，保存并退出此文件。

  输入sudo reboot来重启树莓派。

结果

  重启之后，打开树莓派Linux终端，输入hostname -I，发现此时IP地址已经设置成功~

  笔者一直用的远程连接来连接树莓派，所以能连接上其实已经证明静态IP设置成功了。
