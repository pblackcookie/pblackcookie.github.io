&emsp;&emsp;**笔者经过几天惨痛教训得知一件事——想要修改好主题，找对修改路径是最重要的！！！**
### 为Hexo添加背景图片

&emsp;&emsp;想要添加背景图片，先找到你所选next主题下的`index.styl`文件。笔者选的主题是Gemini,所以笔者的路径就为`blog\themes\next\source\css\_schemes\Gemini\index.styl`。

&emsp;&emsp;打开此文件后，在全文末尾添加如下代码即可。

&emsp;&emsp;不要忘了图片的存储路径是在`blog\themes\next\source\images`下的`background.jpg`哦。

```
body {
    background:url(/images/background.jpg);
    background-repeat: no-repeat;
    background-attachment:fixed; //不重复
    background-size: cover;      //填充
    background-position:50% 50%;
}

```

### 修改Hexo原本主题的透明度

&emsp;&emsp;加上图片后，发现博客的各个框图会挡住图片，那为了露出原本的图，就需要把每一个框都改成略微透明一点才行。

&emsp;&emsp;打开路径为`blog\themes\next\source\css\_schemes\Pisces\_header.styl`的文件，在代码末尾增加如下代码即可。
```
.header-inner{
  opacity: 0.8;
}
```

&emsp;&emsp;打开路径为`blog\themes\next\source\css\_schemes\Gemini\index.styl`的文件，在如下代码中添加一行即可。

```
// Post blocks.
.content-wrap {
  background: initial;
  box-shadow: initial;
  padding: initial;
  opacity: 0.8 //这一行是添加的
}
```

&emsp;&emsp;打开路径为`blog\themes\next\source\css\_schemes\Pisces\_sidebar.styl`的文件，在如下代码中添加一行即可。

```
.sidebar {
  background: var(--body-bg-color);
  box-shadow: none;
  margin-top: 100%;
  position: static;
  width: $sidebar-desktop;
  opacity: 0.8 //这一行是添加的

  +tablet-mobile() {
    display: none;
  }
}
```

### 修改页脚的字
&emsp;&emsp;整个页面添加了背景页脚就看不清了，如图，那这时需要去改变页脚的文字颜色。  

![](https://raw.githubusercontent.com/pblackcookie/ImageHosting/main/test-pic/202208180952672.png)

&emsp;&emsp;惯例先上路径，先查看到页脚模块继承的是`.footer.footer`，在自己博客下路径`blog\themes\next\source\css\_common\outline\footer\footer.styl`中找到`footer.styl`这个文件进行修改。

```
  .footer {
  color: $grey-dark; //修改这里的代码
  font-size: $font-size-small;
  padding: 20px 0;

  &.footer-fixed {
    bottom: 0;
    left: 0;
    position: absolute;
    right: 0;
  }
}
```

&emsp;&emsp;可以看见代码的color这一块是暗灰色`$grey-dark`，只要把它的属性改变就可以改变页脚的颜色了。由于笔者背景图片很花，就将这块的颜色改成深黑色`$black-deep`了。

&emsp;&emsp;修改后的效果如图所示，显而易见地变清楚了很多。  

![](https://raw.githubusercontent.com/pblackcookie/ImageHosting/main/test-pic/202208180956342.png)

&emsp;&emsp;现在用户的图标仿佛格格不入。别着急。打开路径为`blog\themes\next`下的`_config.yml`文件，按下`crtl + F`打开搜索框，输入`copyright`进行跳转，然后修改跳转到这里后代码里的的color属性。

```
  # Icon between year and copyright info.
  icon:
    # Icon name in Font Awesome. See: https://fontawesome.com/icons
    name: fa fa-user
    # If you want to animate the icon, set it to true.
    animated: false
    # Change the color of icon, using Hex Code.
    color: "#222" # 修改这里（#222 是十六进制的深黑色）

  # If not defined, `author` from Hexo `_config.yml` will be used.
  copyright: 来自寒流，爱您。

```

![](https://raw.githubusercontent.com/pblackcookie/ImageHosting/main/test-pic/202208181006433.png)

&emsp;&emsp;修改完毕！
